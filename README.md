# IDC-despertador-inteligente-2026
Este proyecto de Internet de las Cosas (IoT) utiliza una Raspberry Pi Pico y un acelerómetro para diseñar un despertador inteligente capaz de despertarte exclusivamente durante tu fase de sueño ligero, garantizando que empieces el día mucho más descansado y sin la típica fatiga matutina.
